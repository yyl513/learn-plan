# 个人学习成长计划

> 制定时间：2026年4月25日

> 目标周期：GitHub（4周）、Python+VSCode（8周）、英语（12周持续）

---

## 📚 GitHub 学习路径（4周）

### 学习目标
从 GitHub 小白 → 能独立管理仓库、参与开源项目

### 推荐资源

#### 必学基础
| 资源名称 | 类型 | 链接/说明 |
|---------|------|----------|
| Git 官方教程 | 交互式学习 | `git init`、`git add`、`git commit`、`git push` 核心命令 |
| GitHub 官方文档 | 文档 | github.com/git-guides 基础入门 |
| GitHub Skills | 免费课程 | github.com/skills 系列实战课程 |

#### 进阶资料
| 资源名称 | 说明 |
|---------|------|
| 廖雪峰 Git 教程 | 中文友好，适合入门 |
| Git 飞行手册 | 常用命令速查 |
| GitKraken GUI | 可视化工具，辅助理解 |

### 每日任务分配（30分钟）
| 周次 | 重点 | 具体内容 |
|-----|------|---------|
| 第1周 | Git 基础 | 安装 Git、配置用户信息、创建第一个仓库 |
| 第2周 | 版本控制 | 提交历史、分支管理、合并操作 |
| 第3周 | GitHub 协作 | Fork、Pull Request、代码审查流程 |
| 第4周 | 实战应用 | 发布开源项目、配置 Actions CI/CD |

### 实战项目
- 建立个人知识库仓库，记录学习笔记
- 参与一个热门开源项目的文档贡献

## 💻 Python + VSCode 学习路径（8周）

### 学习目标
掌握 Python 基础语法 → 能独立编写小工具/脚本

### 推荐资源

#### 开发环境
| 工具 | 说明 |
|-----|------|
| VSCode | 主力编辑器，必装 Python 插件 |
| Python 3.10+ | 编程语言环境 |
| Git | 版本控制 |

#### 学习资料
| 资源名称 | 类型 | 说明 |
|---------|------|------|
| Python 官方教程 | 文档 | docs.python.org/zh-cn/3/tutorial/ |
| 菜鸟教程 Python | 中文入门 | 简洁易懂 |
| Real Python | 英文进阶 | 深入理解 Python 特性 |
| VSCode Python 教程 | 编辑器 | 官方文档学会用好 VSCode |

#### 辅助工具
| 工具 | 用途 |
|-----|------|
| Black | 代码格式化 |
| Pylint | 代码检查 |
| Jupyter | 数据分析/探索式编程 |

### 每日任务分配（30分钟）
| 周次 | 主题 | 具体内容 |
|-----|------|---------|
| 第1周 | 环境搭建 | 安装 VSCode、Python、配置开发环境 |
| 第2周 | 基础语法 | 变量、数据类型、运算符 |
| 第3周 | 控制流程 | 条件语句、循环语句 |
| 第4周 | 函数 | 函数定义、参数、返回值、作用域 |
| 第5周 | 数据结构 | 列表、元组、字典、集合 |
| 第6周 | 文件操作 | 读写文件、处理 CSV/JSON |
| 第7周 | 模块与包 | 导入模块、pip 包管理 |
| 第8周 | 小项目实战 | 完成一个实用小工具 |

### 推荐实战项目
1. 批量文件重命名脚本
2. 定时任务提醒工具
3. 简单爬虫抓取网页数据

## 🌐 英语学习路径（12周+）

### 学习目标
巩固语法基础 → 扩大词汇量 → 能流畅阅读英文技术文档

### 推荐资源

#### 单词记忆
| 工具/应用 | 说明 |
|----------|------|
| Anki | 间隔重复记忆，定制单词卡片 |
| 墨墨背单词 | 中文用户友好的背词 App |
| Quizlet | 制作学习卡片 |
| Vocabulary.com | 例句丰富，理解性记忆 |

#### 语法学习
| 资源名称 | 说明 |
|---------|------|
| English Grammar in Use | 经典语法书（Raymond Murphy） |
| 赖世雄英语语法 | 中文讲解，易懂 |
| Grammar Girl 播客 | 简短有趣的语法讲解 |

#### 听力口语
| 资源名称 | 说明 |
|---------|------|
| TED Talks | 精选演讲，可调节速度 |
| YouTube Learn English | 多频道可选 |
| 每日英语听力 App | 资源丰富 |

#### 技术英语
| 资源名称 | 说明 |
|---------|------|
| GitHub README | 边学编程边学英语 |
| Stack Overflow | 遇到问题搜英文解答 |
| 技术博客 | Medium、Dev.to |

### 每日任务分配（30分钟）
| 时间段 | 任务 | 时长 |
|-------|------|------|
| 0-10分钟 | 单词复习（Anki/墨墨） | 10分钟 |
| 10-25分钟 | 语法学习/技术文档阅读 | 15分钟 |
| 25-30分钟 | 听力或跟读练习 | 5分钟 |

### 核心原则
- **间隔重复**：使用 Anki 复习旧单词，比死记硬背效率高 3-5 倍
- **语境学习**：通过技术文档、GitHub 项目学单词，在真实语境中记忆
- **主动输出**：尝试用英文写 GitHub 提交信息、README 注释
